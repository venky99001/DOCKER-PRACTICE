- Docker Hub

---

## 📦 1. Docker Hub

### 📌 Definition  
Public registry for Docker images.

### 💻 Command / Example  
`docker pull nginx`

### 🔍 Explanation  
- Stores images  
- Public access  

### 💡 Concept  
- Default registry  

---

## 🐙 2. GitHub Container Registry

### 📌 Definition  
Registry provided by GitHub.

### 💻 Command / Example  
`docker push ghcr.io/user/image`

### 🔍 Explanation  
- Integrated with GitHub  
- Requires authentication  

### 💡 Concept  
- Used in CI/CD  

---

## 🔄 3. Image Push/Pull

### 📌 Definition  
Uploading and downloading images.

### 💻 Command / Example  
`docker push myimage`  
`docker pull myimage`

### 🔍 Explanation  
- push → upload  
- pull → download  

### 💡 Concept  
- Enables sharing
