# 🖼️ Images & Screenshots

## 📂 Organization

Images are organized **day-wise** for better clarity:

* 📁 `day-01/` → Basic concepts screenshots
* 📁 `day-02/` → Docker installation & Nginx
* 📁 `day-03/` → Containers & runtime diagrams
* 📁 More folders will be added as the course progresses

## 🎯 Purpose

These images help to:

* Provide visual understanding of concepts
* Show practical implementation results
* Improve readability of notes
* Help in viva and revision

## 📌 Usage in Notes

Images are used inside markdown files like:

```bash

```
